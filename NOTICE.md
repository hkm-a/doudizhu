# Notices

This project is a fusion effort for a desktop Dou Dizhu game.

## Upstream Projects

- Game foundation: [`svzdev/doudizhu`](https://github.com/svzdev/doudizhu). GitHub license metadata currently reports no license.
- AI target: [`kwai/DouZero`](https://github.com/kwai/DouZero), Apache License 2.0.
- Desktop shell: Tauri 2 and Rust ecosystem dependencies.
- Web client: React, Phaser and related npm dependencies.
- Backend: Tornado, SQLAlchemy and related Python dependencies.

## Bundled Assets

The repository includes static game assets and audio inherited from the game foundation. Their license status follows the same upstream-license caveat described in [LICENSE.md](LICENSE.md).

## Future Work

Before a broad public release, the project should either obtain a clear license for the inherited game foundation or replace the affected code and assets with newly authored or clearly licensed alternatives.
