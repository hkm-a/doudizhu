# Code of Conduct

## Our Standard

这个项目欢迎建设性的技术讨论、清晰的问题报告和善意的不同意见。参与者应当：

- 尊重不同经验水平、语言背景和技术选择。
- 聚焦代码、行为、证据和用户影响。
- 给出可复现信息，避免人身攻击、骚扰或歧视。
- 在分歧中尽量提出可验证的下一步。

## Unacceptable Behavior

- 侮辱、威胁、骚扰或持续打断他人。
- 发布他人的隐私信息、密钥或未授权数据。
- 故意提交恶意代码、后门、误导性 issue 或垃圾信息。
- 在安全问题修复前公开可利用细节。

## Enforcement

维护者可以编辑、隐藏或删除违反本准则的评论、issue、PR 或提交，并在必要时限制参与。严重安全或滥用行为会被优先处理。
